ReactStruct

The ReactStruct is a software application designed by ReactionTest to simplify complex calculations and reduce errors in structural engineering projects. This tool provides engineers and designers with an intuitive interface to perform various calculations related to structural analysis and design.

Download

Tools

Slab Load on Beam

Calculate UDL on beam distributed by slab from One & Two way slabs

Time Period

Calculate the Time Period of RCC structures for All Cases as per IS 1893:2016

Base Shear

Base Shear calculator with automated Sa/g calculation for Equivalent Static Method and Response Spectrum Method

Beam Partial Release

Easily calculate the reduced stiffness of frame members for ETABS to optimize design and achieve desired behaviour

Seismic Weight Calculator

Quick Seismic weight calculator for Base Shear

2. Run the installer to automatically set up the necessary dependencies. You can create shortcut on desktop for easy access to the application.

3. You will be presented with the Home Page of ReactStruct after successful installation.
